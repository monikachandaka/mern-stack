HTML Tags:
----------
    ->tags are the predefined words which enclosed with both opening and closing angulaer brackets

    ->ex: <h1></h1>,<div></div>,<head></head>


Types of tags:
--------------
    1. Paired tags
            -> any tags having both opening and closing tags
            ex:<html></html>, <head></head>........etc

            i. heading tags

    
            ii. formatting tags
                    ->these tags have thir own CSS behaviors
                    Ex:<b></b>

            iii. other tags
                    <div></div>, <p</p>, <span></span>, list tags , semantic tags, media tags, table tags , form tags....

    2. Unpaired tags:
            ->any tags having onlu=y opening tags is called as a upaired tags 
            Ex : <br>, <input>, <hr>, <meta> , <col>.....etc


Attributes in HTML:
-------------------
        -> Attributes are used to give additional information to the html tags
        -> Attributes are always in opening tags
        ->most of the HTML elements having Attributes
        -> attributes are always key and value Paired

        ex:<h1 align="center">HELLO</h1>

ELEMENTS IN HTML:
----------------
        ->The opening tag , closing tag and content together is called as a HTML elements
          Ex:<p>This is a paragraph</p>

          Types of elements:
                1. Inline level elements
                        ->the elements that only consumes only content width
                         EX: All formatting tags, <span></span>......etc
                2. block level elements
                        ->the elements which consume entire width of the browsers
                        Ex: all heading tags, <div></div>, other tags....etc
