JSX(javascript and XML[extendable markup language]):
---------------------------------------------------
     ->React will understand JSX code but browsers will not understand JSX it understand only JS code(because every browsers having JS engine's)
     ->to convert JSX code to JS(javascript) we have a next generation compiler called "babel JS"
   
  ->JSX is similar to HTML , here it allows both JS and HTML code


  Rules:
  --------
      ->JSX filenames always starts with capital letter
      ->Here we can achieve javascript expression -> {}
      ->IN return function or values HTML elements contains parentTag
      ->JSX will allow empty fragments (<> </>)
